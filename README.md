# Conexão com o banco

### 1. Configurar Variáveis de Ambiente
Crie um arquivo `.env` na raiz com:

```env
DB_HOST=localhost
DB_USER=seu_usuario
DB_PASSWORD=senha  
DB_NAME=nome_banco
DB_PORT=3306
```

### 2. Execução do Projeto
Instalação e inicialização

- cd backend
- npm install
- node index.js

### 3. Execução do Frontend
Na raiz do projeto, execute:
- npm start
